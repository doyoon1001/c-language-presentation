팀명 반대표와 아이들 — 인덕대학교 C언어 팀 프로젝트 · 2026년도 1학기

Problem Definition

문제 정의

문제 목표

사용자로부터 여러 과목의 점수를 입력받아
성적 평균을 계산한 뒤 화면에 출력하는 프로그램을 작성한다.

📥 각 과목 점수 입력 → 합계 누적
➗ 평균 = 합계 ÷ 과목 수
사용 헤더
#include <stdio.h> — printf, scanf 사용
입력 (Input)

과목 수 n과 각 과목의 점수를 반복 입력

scanf("%d", &score);
처리 (Process)

합계 누적

sum += score

평균 계산

avg = sum / n
출력 (Output)

계산된 성적 평균을 printf로 화면에 출력

성적 평균: %.2f
Flowchart

순 서 도

시 작
초기화
sum = 0, i = 0
입력
과목 수 n 입력
조건
i < n ?
처리
점수 입력, sum += score, i++
출력
평균 = sum / n, 화면 출력
끝
입력 단계

과목 수 n을 먼저 입력받고, 반복문을 돌며 각 과목 점수를 scanf로 입력받는다.

반복 단계 — 합계 누적

🔁 반복 조건

for (i = 0; i < n; i++)

➕ 합계 누적

sum += score
출력 단계

반복 종료 후 평균 = sum / n 계산 결과를 printf로 출력한다.

Code Review

성적 평균 코드 리뷰

변수 선언

int형 변수 n(과목 수), score(점수), sum(합계), i(반복 인덱스)를 선언합니다.

입력 처리

scanf("%d", &score) — 반복문 안에서 과목 점수를 순서대로 입력받습니다.

핵심 로직
  • for 반복문 — i가 0부터 n-1까지 순회
  • sum += score — 점수를 합계에 누적
  • sum / n — 최종 평균 계산 후 출력
Execution Result

실 행 결과

Terminal — 실행 결과
결과 분석

입력한 각 과목 점수가 반복문을 통해 누적 합산되고, 최종적으로 과목 수로 나눈 성적 평균이 올바르게 출력됩니다.

핵심 포인트
  • 반복문으로 여러 값을 순차 입력
  • 누적 변수(sum)로 합계를 쌓아가는 패턴
  • 정수 나눗셈 주의 — 평균은 실수형으로 출력
Reflections

팀 장

평균을 계산할 때 3이 아닌 3.0으로 나눠야 소수점 결과가 제대로 나온다는 점이 인상적이었습니다. 정수끼리의 나눗셈은 소수점이 버려지기 때문에 float형 연산임을 명시하는 것이 중요하다는 걸 배웠습니다.
