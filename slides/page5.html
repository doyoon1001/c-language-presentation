05. BASIC STRUCTURE

C언어 프로그램의 기본 구조

Standard C Program Template

// "Hello, World!" 출력 예제

#include <stdio.h>

int main() {

printf("Hello, World!\n");

return 0;

}

* C언어는 함수들의 집합으로 구성됩니다.

#include <stdio.h>

표준 입출력 함수(printf 등)가 정의된 헤더 파일을 포함하여 기능을 확장합니다.

int main()

프로그램이 실행될 때 가장 먼저 호출되는 시작점(Entry Point) 함수입니다.

printf() 함수

괄호 안의 내용을 화면에 출력하는 표준 출력 함수입니다. (\n은 줄바꿈)

return 0;

성공적으로 종료되었음을 운영체제에 알리고 함수를 종료합니다.

모든 C 프로그램은 반드시 하나 이상의 main 함수를 포함해야 실행 가능합니다.
