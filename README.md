# C언어 팀플 발표 자료

## 🚀 개요
본 프로젝트는 C언어의 역사, 특징, 장단점, 그리고 현대 IT 산업에서의 중요성을 다루는 발표 자료입니다. 4명의 팀원이 협업하여 제작했습니다.

## 📂 프로젝트 구조
- `index.html`: 모든 슬라이드를 화면에 띄우는 메인 통합 뷰어
- `slides/chapter1_slides_new.html`: C언어 발표 내용과 애니메이션이 포함된 전체 슬라이드 파일
- `src/concat.js`: (이전 버전 호환용) 방향키 슬라이드 이동 제어 스크립트

## 🛠️ 사용 방법

### 슬라이드 실행 방법
발표 자료를 보려면 최상단 디렉토리의 `index.html`을 실행하면 됩니다.
```bash
open index.html
```
브라우저에서 `index.html`이 열리면, **키보드 방향키(`←`, `→`)**를 누르거나 화면 우측 하단의 네비게이션 버튼을 클릭하여 슬라이드를 넘길 수 있습니다. `F11`을 누르거나 상단의 **전체화면(⛶)** 버튼을 클릭하여 전체화면으로 발표를 진행할 수 있습니다.

## 🎨 기술 스택
- **HTML5**: 웹 표준 기반 슬라이드 제작
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크
- **Font Awesome**: 아이콘 라이브러리
- **JavaScript**: 슬라이드 합치기 기능
