# C언어 팀플 발표 자료

## 🚀 개요
본 프로젝트는 C언어의 역사, 특징, 장단점, 그리고 현대 IT 산업에서의 중요성을 다루는 발표 자료입니다. 4명의 팀원이 협업하여 제작했습니다.

## 📂 프로젝트 구조
- `index.html`: 모든 슬라이드를 한 번에 이어서 볼 수 있는 통합 뷰어 화면 (메인 실행 파일)
- `slides/page1.html` ~ `page9.html`: 각 발표 슬라이드를 구성하는 개별 HTML 파일
- `src/concat.js`: 개별 슬라이드 페이지에서 키보드 방향키 이동을 지원하는 제어 스크립트

## 🛠️ 사용 방법

### 1. 통합 뷰어로 전체 슬라이드 보기 (권장)
가장 편리하게 슬라이드를 보는 방법은 최상단 디렉토리의 `index.html`을 실행하는 것입니다.
```bash
open index.html
```
브라우저에서 `index.html`을 연 후, 화면 하단이나 **키보드 방향키(`←`, `→`), 스페이스바**를 이용하여 발표를 진행할 수 있습니다.

### 2. 개별 슬라이드 보기
필요한 경우 `slides/` 폴더 내의 각 HTML 파일을 독립적으로 실행할 수도 있습니다.
```bash
open slides/page1.html
open slides/page2.html
...
```
개별 페이지에서도 `src/concat.js` 스크립트에 의해 **키보드 방향키(`←`, `→`)**를 누르면 다음 슬라이드로 넘어갑니다.

## 🎨 기술 스택
- **HTML5**: 웹 표준 기반 슬라이드 제작
- **Tailwind CSS**: 유틸리티 우선 CSS 프레임워크
- **Font Awesome**: 아이콘 라이브러리
- **JavaScript**: 슬라이드 합치기 기능
